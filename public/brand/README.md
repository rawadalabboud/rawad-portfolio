# rawad.ai — Noir Atelier Brand

## Concept
Editorial luxury tech portfolio. Ember coral + aurora violet on deep noir.

## Mark: **ra pulse**
- **`ra` ligature** — Rawad + rawad.ai domain
- **Open loop** — the `a` ring suggests continuous systems
- **Pulse crest** — waveform above the letterform (EEG, voice, streaming). Signal processing roots, not a generic orbit
- **Ember hub** — input / orchestration point on the stem
